The following files contain SAS code that create examples
of dashboard indicators or applications.  For more information,
see

   http://support.sas.com/rnd/datavisualization/dashboards/

 ------------------

sliderInd.sas       - slider chart indicator (SAS program version)
sliderIndSTP.sas    - slider chart indicator (stored process version)
sliderIndPortal.sas - slider chart indicator (portlet version)

sliderDash.sas       - slider chart dashboard (SAS program version)
sliderDashSTP.sas    - slider chart dashboard (stored process version)
sliderDashPortal.sas - slider chart dashboard (portlet version) 

bulletGraphInd.sas       - bullet graph indicator (SAS program version) 
bulletGraphIndSTP.sas    - bullet graph indicator (stored process version) 
bulletGraphIndPortal.sas - bullet graph indicator (portlet version) 

bulletGraphDash.sas       - bullet graph dashboard (SAS program version)
bulletGraphDashSTP.sas    - bullet graph dashboard (stored process version)
bulletGraphDashPortal.sas - bullet graph dashboard (portlet version) 

dialMeterInd.sas       - dial meter indicator (SAS program version)
dialMeterIndSTP.sas    - dial meter indicator (stored process version)
dialMeterIndPortal.sas - dial meter indicator (portlet version) 

dialMeterDash.sas       - dial meter dashboard (SAS program version)
dialMeterDashSTP.sas    - dial meter dashboard (stored process version)
dialMeterDashPortal.sas - dial meter dashboard (portlet version) 

barChartInd.sas       - bar chart indicator (SAS program version)
barChartIndSTP.sas    - bar chart indicator (stored process version)
barChartIndPortal.sas - bar chart indicator (portlet version) 

barChartDash.sas       - bar chart dashboard (SAS program version)
barChartDashSTP.sas    - bar chart dashboard (stored process version)
barChartDashPortal.sas - bar chart dashboard (portlet version) 

telesalesDash.sas       - telesales dashboard (SAS program version)
telesalesDashSTP.sas    - telesales dashboard (stored process version)
telesalesDashPortal.sas - telesales dashboard (portlet version) 

webMarketDash.sas       - Web barketing dashboard (SAS program version)
webMarketDashSTP.sas    - Web barketing dashboard (stored process version)
webMarketDashPortal.sas - Web barketing dashboard (portlet version)

copyToWebDAV.sas - SAS macro code to add the directory that contains the
                   portlet version of the samples to a WebDAV repository

generic_drilldown.htm - Default target page for feature drill-down links

 grdbrd.pdf - Documentation that explains how to implement dashboard 
              applications using these samples
